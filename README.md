## Quick Start

### 1. Clone and Setup

```bash
cd /home/super/Documents/QuizAppFull
```

### 2. Create Virtual Environment (Already done)

The project comes with a pre-configured virtual environment.

### 3. Install Dependencies

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

### 4. Setup Database

```bash
./dev.sh migrate
```

### 5. Start Development Servers

You need to run both Django and Tailwind in separate terminals:

**Terminal 1 - Django Server:**
```bash
./dev.sh runserver
```

**Terminal 2 - Tailwind CSS:**
```bash
./dev.sh tailwind
```

The application will be available at `http://127.0.0.1:8000/`

## Development Helper Script

The `dev.sh` script provides convenient commands for development:

```bash
./dev.sh runserver       # Start Django development server
./dev.sh tailwind        # Start Tailwind CSS watch mode
./dev.sh build          # Build Tailwind CSS for production
./dev.sh migrate        # Run database migrations
./dev.sh makemigrations # Create new migrations
./dev.sh collectstatic  # Collect static files
./dev.sh createsuperuser # Create admin user
./dev.sh shell          # Start Django shell
./dev.sh test           # Run tests
```

## Production Deployment

For production deployment, use the simplified `prod.sh` script:

### Quick Production Start

```bash
# Deploy with all steps (build, migrate, collect static, start server)
./prod.sh deploy

# Start on different host/port with more workers
./prod.sh deploy 0.0.0.0 8080 4
```

### Individual Commands

```bash
./prod.sh build       # Build Tailwind CSS only
./prod.sh migrate     # Run database migrations only
./prod.sh static      # Collect static files only
./prod.sh start       # Start server only (on 127.0.0.1:8000)
```

### Production Features

- **Uvicorn ASGI server** for high performance
- **WhiteNoise** for efficient static file serving
- **Minified Tailwind CSS** with daisyUI
- **Production-optimized Django settings**
